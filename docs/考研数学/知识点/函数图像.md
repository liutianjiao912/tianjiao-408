## 一、六大基本初等函数图像（“常反对幂指三”）

这是考研数学中最基础、最常考的一类函数图像，几乎每年都会涉及。

### 1. **常数函数**
- 形式：$y = C$
- 图像：平行于$x$轴的直线
- 特点：无增减性，导数恒为0

### 2. **幂函数**
- 形式：$y = x^a$（$a \in \mathbb{R}$）
- 常见情形：
  -$a > 0$：过原点，单调递增（如$y = x^2, x^{1/2}$）
  -$a < 0$：在$(0, +\infty)$单调递减（如$y = x^{-1}, x^{-2}$）
- 注意奇偶性和定义域变化

### 3. **指数函数**
- 形式：$y = a^x$（$a > 0, a \ne 1$）
- 图像特征：
  -$a > 1$：单调递增，过点$(0,1)$
  -$0 < a < 1$：单调递减
- 极限行为：$\lim_{x \to +\infty} a^x = +\infty$（当$a>1$）；$\lim_{x \to -\infty} a^x = 0$

### 4. **对数函数**
- 形式：$y = \log_a x$
- 是指数函数的反函数，图像关于$y=x$对称
- 定义域$(0, +\infty)$，值域$\mathbb{R}$
-$a > 1$时单调递增；$0<a<1$时单调递减

### 5. **三角函数**
包括以下六种（统称“三角六函数”）：
- **正弦函数**：$y = \sin x$，周期$2\pi$，值域$[-1,1]$，奇函数
- **余弦函数**：$y = \cos x$，周期$2\pi$，偶函数
- **正切函数**：$y = \tan x$，周期$\pi$，在$(-\frac{\pi}{2}+k\pi, \frac{\pi}{2}+k\pi)$内单调递增
- **余切函数**：$y = \cot x$，周期$\pi$，在$(k\pi, (k+1)\pi)$内单调递减
- **正割函数**：$y = \sec x = 1/\cos x$
- **余割函数**：$y = \csc x = 1/\sin x$

### 6. **反三角函数**
- **反正弦**：$y = \arcsin x$，定义域$[-1,1]$，值域$[-\frac{\pi}{2}, \frac{\pi}{2}]$，单调递增
- **反余弦**：$y = \arccos x$，值域$[0, \pi]$，单调递减
- **反正切**：$y = \arctan x$，定义域$\mathbb{R}$，值域$(-\frac{\pi}{2}, \frac{\pi}{2})$，单调递增
- **反余切**：$y = \operatorname{arccot} x$，值域$(0, \pi)$，单调递减

## 二、常见复合与特殊函数图像

这些函数虽非基本初等函数，但在极限、连续性、可导性等考点中频繁出现。

### 1. **震荡型函数**
-$y = \sin(1/x)$：在$x \to 0$时无限震荡，无极限，但有界（值域仍为$[-1,1]$）
-$y = x\sin(1/x)$：在$x \to 0$时极限为0（无穷小 × 有界）

### 2. **重要极限相关函数**
-$y = \frac{\sin x}{x}$：偶函数，在$x=0$处无定义但极限为1；$x \to \infty$时趋于0
-$y = \left(1 + \frac{1}{x}\right)^x$：单调递增趋于$e$，是自然对数底数的定义式

### 3. **绝对值函数**
-$y = \lvert x \rvert$：V形图像，不可导点在$x=0$，常用于分段讨论

### 4. **符号函数与取整函数**
- 符号函数：$y = \operatorname{sgn}(x) = \begin{cases} 1 & x>0 \\ 0 & x=0 \\ -1 & x<0 \end{cases}$
- 取整函数（高斯函数）：$y = [x]$，阶梯状图像，处处不连续（除整数点外）

### 5. **狄利克雷函数**
-$y = \begin{cases} 1, & x \in \mathbb{Q} \\ 0, & x \notin \mathbb{Q} \end{cases}$
- 处处不连续、不可积（黎曼意义下），用于反例构造

## 三、参数方程与极坐标曲线（常考面积、弧长）

这些图像多出现在多元微积分或几何应用题中：

- **心形线**：$r = a(1 + \cos\theta)$
- **星形线**：$x^{2/3} + y^{2/3} = a^{2/3}$
- **摆线（旋轮线）**：$x = a(t - \sin t),\ y = a(1 - \cos t)$
- **玫瑰线**：$r = a \cos(n\theta)$或$\sin(n\theta)$
- **伯努利双纽线**：$(x^2 + y^2)^2 = a^2(x^2 - y^2)$

## 四、双曲函数及其反函数（部分院校数一、数三涉及）

- **双曲正弦**：$y = \sinh x = \frac{e^x - e^{-x}}{2}$，奇函数，单调递增
- **双曲余弦**：$y = \cosh x = \frac{e^x + e^{-x}}{2}$，偶函数，最小值为1
- **双曲正切**：$y = \tanh x = \frac{\sinh x}{\cosh x}$，值域$(-1,1)$
- 反双曲函数如$y = \operatorname{arsinh} x = \ln(x + \sqrt{x^2 + 1})$等

---

## 五、其他高频函数

- **分式函数**：如$y = \frac{x+1}{x-1}$，有垂直渐近线$x=1$和水平渐近线$y=1$
- **指数衰减/增长组合**：如$y = e^{1/x}$，在$x \to 0^+$时趋于$+\infty$，$x \to 0^-$时趋于0
