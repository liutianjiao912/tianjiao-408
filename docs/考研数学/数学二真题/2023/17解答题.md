## 🎯 题目解析：微分方程几何应用

**题目回顾**：
设曲线 $L:y=y(x)\;(x>e)$ 经过点 $(e^2,0)$，$L$ 上任一点 $P(x,y)$ 到 $y$ 轴的距离等于该点处的切线在 $x$ 轴上的截距。
(1) 求 $y(x)$；
(2) 求面积最小值的切线点及最小面积。

---

### ⚠️ 题目勘误与假设分析

在开始解答前，我们需要先审视题目中的几何条件。
* **条件 A**：点 $P(x,y)$ 到 $y$ 轴的距离为 $|x|$（因 $x>e$，即为 $x$）。
* **条件 B**：切线在 $x$ 轴上的截距公式为 $x - \frac{y}{y'}$。

若严格按照字面意思“距离等于 $x$ 轴截距”，即 $x = x - \frac{y}{y'}$，这将直接导致 $\frac{y}{y'} = 0 \implies y=0$。这是一个平凡解（直线），且通常不符合此类考研数学考试题考察微分方程求解的初衷。

**最可能的原题意图**：
根据此类经典试题的结构，题目极大概率存在笔误，正确的几何条件通常是以下两种情况之一：
1.  到 $y$ 轴的距离等于切线在 **$y$ 轴**上的截距。
2.  到 $x$ 轴的距离等于切线在 $x$ 轴上的截距（但这会导致 $y=0$ 的奇点问题）。

结合 $y(e^2)=0$ 这一边界条件，**情况 1（切线在 $y$ 轴截距）** 能得出一个优美的非平凡解。
**因此，本题将按照“点 $P$ 到 $y$ 轴的距离等于该点处的切线在 $y$ 轴上的截距”进行解答。**

---

## 第一部分：求曲线方程 $y(x)$

### 1.1 建立微分方程

设曲线上的点为 $P(x, y)$。
1.  **点 $P$ 到 $y$ 轴的距离**：
    由于 $x > e > 0$，距离 $d = x$。

2.  **切线在 $y$ 轴上的截距**：
    曲线在点 $P(x,y)$ 处的切线方程为：
    $$Y - y = y'(X - x)$$
    令 $X = 0$ 求 $y$ 轴截距 $Y_{int}$：
    $$Y_{int} - y = y'(0 - x) \implies Y_{int} = y - xy'$$

3.  **根据题意建立等式**：
    $$x = y - xy'$$
    整理得一阶线性微分方程：
    $$xy' - y = -x \implies y' - \frac{1}{x}y = -1$$

### 1.2 求解微分方程

这是一个形如 $y' + P(x)y = Q(x)$ 的一阶线性微分方程，其中 $P(x) = -\frac{1}{x}$，$Q(x) = -1$。

1.  **计算积分因子**：
    $$I(x) = e^{\int P(x)dx} = e^{\int -\frac{1}{x}dx} = e^{-\ln x} = \frac{1}{x}$$

2.  **通解公式**：
    $$y \cdot \frac{1}{x} = \int (-1) \cdot \frac{1}{x} \, dx = -\int \frac{1}{x} \, dx$$
    $$\frac{y}{x} = -\ln x + C$$
    $$y = x(C - \ln x)$$

### 1.3 代入初始条件确定常数

曲线经过点 $(e^2, 0)$，即当 $x = e^2$ 时，$y = 0$。
代入通解：
$$0 = e^2(C - \ln e^2)$$
$$0 = e^2(C - 2)$$
因为 $e^2 \ne 0$，故解得 $C = 2$。

✅ **曲线方程为**：
$$y = x(2 - \ln x) \quad (x > e)$$

## 第二部分：面积最小化问题

### 2.1 确定切线与坐标轴的交点

设切点为 $P(x_0, y_0)$，其中 $y_0 = x_0(2 - \ln x_0)$。
切线方程为 $Y - y_0 = y_0'(X - x_0)$。

首先求导数 $y'$：
$$y' = (2x - x\ln x)' = 2 - (1 \cdot \ln x + x \cdot \frac{1}{x}) = 2 - (\ln x + 1) = 1 - \ln x$$

1.  **切线在 $y$ 轴的截距 ($Y_{int}$)**：
    根据题意（或微分方程条件），已知 $Y_{int} = x_0$。
    *(验证：$y_0 - x_0 y_0' = x_0(2-\ln x_0) - x_0(1-\ln x_0) = 2x_0 - x_0\ln x_0 - x_0 + x_0\ln x_0 = x_0$。正确)*

2.  **切线在 $x$ 轴的截距 ($X_{int}$)**：
    令 $Y=0$，解 $X$：
    $$0 - y_0 = y_0'(X - x_0)$$
    $$X = x_0 - \frac{y_0}{y_0'} = x_0 - \frac{x_0(2 - \ln x_0)}{1 - \ln x_0}$$
    $$X = x_0 \left( 1 - \frac{2 - \ln x_0}{1 - \ln x_0} \right) = x_0 \left( \frac{1 - \ln x_0 - 2 + \ln x_0}{1 - \ln x_0} \right)$$
    $$X = x_0 \left( \frac{-1}{1 - \ln x_0} \right) = \frac{x_0}{\ln x_0 - 1}$$

### 2.2 建立面积函数

切线与两坐标轴围成的三角形面积 $S$ 为：
$$S(x) = \frac{1}{2} |X_{int}| \cdot |Y_{int}| = \frac{1}{2} \left| \frac{x}{\ln x - 1} \right| \cdot |x|$$

由于题目给定 $x > e$，故 $\ln x > 1$，$\ln x - 1 > 0$，$x > 0$。绝对值符号可直接去掉：
$$S(x) = \frac{x^2}{2(\ln x - 1)}$$

### 2.3 求极值

对 $S(x)$ 求导以寻找驻点：
$$S'(x) = \frac{1}{2} \cdot \frac{(x^2)'(\ln x - 1) - x^2(\ln x - 1)'}{(\ln x - 1)^2}$$
$$S'(x) = \frac{1}{2} \cdot \frac{2x(\ln x - 1) - x^2(\frac{1}{x})}{(\ln x - 1)^2}$$
$$S'(x) = \frac{1}{2} \cdot \frac{2x\ln x - 2x - x}{(\ln x - 1)^2} = \frac{x(2\ln x - 3)}{2(\ln x - 1)^2}$$

令 $S'(x) = 0$，得：
$$2\ln x - 3 = 0 \implies \ln x = \frac{3}{2} \implies x = e^{3/2}$$

验证极值性质：
* 当 $e < x < e^{3/2}$ 时，$2\ln x - 3 < 0$，即 $S'(x) < 0$，函数递减。
* 当 $x > e^{3/2}$ 时，$2\ln x - 3 > 0$，即 $S'(x) > 0$，函数递增。
* 因此，$x = e^{3/2}$ 是唯一的极小值点，也是最小值点。

### 2.4 计算坐标与最小面积

1.  **求切点坐标**：
    将 $x = e^{3/2}$ 代入 $y(x) = x(2 - \ln x)$：
    $$y = e^{3/2} \left( 2 - \frac{3}{2} \right) = \frac{1}{2}e^{3/2}$$
    即切点为 $(e^{3/2}, \frac{1}{2}e^{3/2})$，或写为 $(e\sqrt{e}, \frac{1}{2}e\sqrt{e})$。

2.  **求最小面积**：
    代入面积公式 $S(x)$：
    $$S_{\min} = S(e^{3/2}) = \frac{(e^{3/2})^2}{2(\frac{3}{2} - 1)} = \frac{e^3}{2 \cdot \frac{1}{2}} = e^3$$

---

## 📝 总结答案

1.  曲线 $L$ 的方程为：
    $$y = x(2 - \ln x), \quad x > e$$

2.  使面积最小的切点坐标为 **$(e^{3/2}, \frac{1}{2}e^{3/2})$**，此时最小面积为 **$e^3$**。
