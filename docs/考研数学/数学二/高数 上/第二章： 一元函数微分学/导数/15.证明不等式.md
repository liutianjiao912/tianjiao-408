# 二、知识小结：含未知数不等式的证明方法与函数选择题技巧

## 一、不等式证明的两类题型及通用策略

### ✅ 类型一：不等式含两个未知数 $x_1, x_2$

#### 核心思想
当不等式中包含两个变量时，通常通过**构造函数、分离变量、比较函数值大小**来证明。

#### 解题步骤（四步法）

1. **处理底数和指数均为未知数的部分**

   将形如 $a^b$ 的表达式转化为以 $e$ 为底的形式：
   $$
   a^b = e^{b \ln a}
   $$

   利用指数函数单调性：
   $$
   e^m > e^n \iff m > n
   $$

   即可将指数不等式转化为对数形式的代数不等式，消去 $e$。

2. **将同一未知数集中到一边**

   在不等式两边同时除以一个大于零的未知数（注意符号判断），使同类项合并。

3. **设函数 $f(x)$：构造相同结构函数**

   若不等式两边具有相似结构（如都含 $x_1, x_2$ 的指数或对数），设：
   $$
   f(x) = \text{该相同结构}
   $$

   例如：
   $$
   \text{若要证 } a^{x_1} > a^{x_2},\quad \text{设 } f(x) = a^x
   $$

4. **比较 $f(x_1)$ 与 $f(x_2)$ 的大小**

   在给定 $x_1, x_2$ 的范围内，分析 $f(x)$ 的单调性，进而比较函数值大小。

> 📌 示例：
> 要证：$x_1^2 < x_2^2$，且 $0 < x_1 < x_2$  
> 构造 $f(x) = x^2$，则 $f'(x) = 2x > 0$（当 $x > 0$），故 $f(x)$ 单调递增，因此 $f(x_1) < f(x_2)$

---

### ✅ 类型二：不等式只含一个未知数 $x$

#### 核心思想
通过**构造函数、化为与零的关系、求导分析单调性**来证明不等式成立。

#### 解题步骤（四步法）

1. **处理底数和指数均为未知数的部分**

   同样将 $a^{x+b}$ 转化为：
   $$
   a^{x+b} = e^{(x+b)\ln a}
   $$

   再利用 $e^m > e^n \iff m > n$ 消去 $e$。

2. **将式子化为与零的关系，并设函数 $f(x)$**

   把不等式变形为“某式 $\geq 0$”或“某式 $\leq 0$”，令：
   $$
   f(x) = \text{该式子中与 } \ln \text{ 相关的部分}
   $$

   例如：
   $$
   \text{若要证 } x \ln x \geq 0,\quad \text{设 } f(x) = x \ln x
   $$

3. **设 $f(\alpha) = 0$，求解 $\alpha$**

   一般先尝试代入 $\alpha = 0$ 验证是否满足；
   若 $f(\alpha) = 0$，则可研究 $f(x)$ 在 $x > \alpha$ 或 $x < \alpha$ 上的符号。

4. **比较 $f(x)$ 与 $f(\alpha)$ 的大小**

   在给定区间内（如 $x > 0$）分析 $f(x)$ 的单调性，判断其与 $f(\alpha)$ 的关系。

> 📌 示例：
> 要证：$x \ln x \geq 0$ 对所有 $x \geq 1$ 成立  
> 设 $f(x) = x \ln x$，$f(1) = 0$  
> $f'(x) = \ln x + 1 > 0$ 当 $x > 1/e$，故 $f(x)$ 在 $[1,+\infty)$ 上单调递增  
> 所以 $f(x) \geq f(1) = 0$

## 二、考试重点与易混淆点

### ✅ 考试重点

| 题型 | 考查内容 | 分值 |
|------|----------|-------|
| **含两个未知数的不等式证明** | 掌握每一步转化方法和函数构造，常以解答题出现 | 8–15 分 |
| **含一个未知数的不等式证明** | 构造函数并求解 $\alpha$，常以解答题出现 | 6–12 分 |
| **含 $f(x)$ 的选择题** | 熟悉常见函数求导公式，判断单调性选答案 | 4–6 分 |

---

### ❌ 易混淆点

1. **两种不等式证明的区别**
   - 含两个未知数 → **分离变量、构造相同结构函数**；
   - 含一个未知数 → **化为与零的关系、构造函数比较**。

2. **构造函数的方式不同**
   - 两个未知数：设 $f(x)$ 为相同结构；
   - 一个未知数：设 $f(x)$ 为整个式子或与 $\ln$ 相关部分。

3. **求导公式的误用**
   - 如 $(\ln x)' = \frac{1}{x}$，$(x \ln x)' = \ln x + 1$；
   - 混淆 $f(x)$ 与 $f'(x)$ 的作用；
   - 不同函数混合时，未正确应用复合函数求导法则。

4. **单调性判断错误**
   - 忽略定义域限制（如 $x > 0$）；
   - 误判 $f'(x)$ 正负导致结论相反。

## 三、含 $f(x)$ 的选择题解法技巧

### ✅ 解题思路

这类题目通常给出一个函数 $f(x)$，要求判断其性质（如单调性、极值、图像等），或从选项中选出正确的结论。

#### 步骤一：分析式子是哪一个式子的导数

- 若只涉及 $f(x)$，查找仅含 $f(x)$ 的求导公式；
- 若既含 $f(x)$ 又含 $g(x)$，查找包含加减乘除的求导公式。

> 常见导数公式回顾：
- $(f+g)' = f' + g'$
- $(fg)' = f'g + fg'$
- $\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$
- $(f(g(x)))' = f'(g(x)) \cdot g'(x)$

---

#### 步骤二：分析导数正负，得出单调性

- 若 $f'(x) > 0$，则 $f(x)$ 单调递增；
- 若 $f'(x) < 0$，则 $f(x)$ 单调递减；
- 结合选项，依据单调性选择正确答案。

> 🔍 示例：
> 已知 $f'(x) = x^2 - 1$，问 $f(x)$ 在哪个区间单调递增？
>
> 解：$f'(x) > 0 \Rightarrow x^2 - 1 > 0 \Rightarrow x < -1$ 或 $x > 1$
>
> 故 $f(x)$ 在 $(-\infty, -1)$ 和 $(1, +\infty)$ 上单调递增。

## 四、总结口诀

> **“双未知，构函数，结构相同才好比；  
> 单未知，化为零，比较大小看单调；  
> 导数正，函数增；导数负，函数减；  
> 求导准，判断清，选择题里稳拿分。”**

### 🧠 学习建议

1. **熟练掌握指数与对数的转换技巧**：$a^b = e^{b \ln a}$ 是关键工具；
2. **加强函数构造训练**：学会从不等式中提取“结构”来设函数；
3. **熟悉常见函数的导数公式**，尤其是复合函数；
4. **练习分类讨论**：区分“两个未知数”与“一个未知数”的处理方式；
5. **注意定义域和边界条件**，避免逻辑漏洞。
