

## 一、第二类换元法的基本概念与使用场景

### ✅ 核心内容

**适用情况**：  
当被积函数中含有形如 $\sqrt{ax^2 + bx + c}$ 的根号表达式时，且该二次式可通过配方法转化为以下三种标准形式之一：

1. $\sqrt{(x + c)^2 + a^2}$
2. $\sqrt{(x + c)^2 - a^2}$
3. $\sqrt{a^2 - (x + c)^2}$

此时可使用**第二类换元法**（即**三角代换法**）进行积分。

---

### ✅ 常见变形示例

- $\sqrt{x^2 - 2x} = \sqrt{(x - 1)^2 - 1}$
- $\sqrt{x^2 + 1} = \sqrt{(x + 0)^2 + 1^2}$

> 🔹 提示：先通过**配方**将根号内化为标准形式。

---

### ✅ 考试重点 / 易混淆点

- **重点**：识别根号内的结构，判断应使用哪种三角代换；
- **易错点**：
  - 混淆不同形式的代换方式；
  - 忽略配方步骤，直接代换导致错误；
  - 可能以选择题或计算题形式考查，分值约 5–10 分。

---

## 二、第二类换元法解题步骤（示例一）

### ✅ 解题流程

#### 第一步：设 $X = x + c$，$dX = dx$

- 目的是简化根号内的表达式；
- 例如：若原式含 $\sqrt{(x - 1)^2 - 1}$，则设 $X = x - 1$，$dX = dx$

#### 第二步：根据根号内形式选择三角代换

| 根号内形式 | 代换方式 | $dX$ 表达式 |
|-----------|----------|-------------|
| $X^2 + a^2$ | $X = a \tan t$ | $dX = a \sec^2 t\,dt$ |
| $X^2 - a^2$ | $X = a \sec t$ | $dX = a \sec t \tan t\,dt$ |
| $a^2 - X^2$ | $X = a \sin t$ | $dX = a \cos t\,dt$ |

#### 第三步：化简并求积分

- 将原积分转换为关于 $t$ 的积分；
- 利用三角恒等式化简；
- 积分后将结果还原为 $x$ 的表达式。

---

### ✅ 示例：计算 $\int \frac{dx}{\sqrt{(x - 1)^2 - 1}}$

**解**：

1. 设 $X = x - 1$，则 $dX = dx$，原式变为：
   $$
   \int \frac{dX}{\sqrt{X^2 - 1}}
   $$

2. 因含 $\sqrt{X^2 - 1}$，设 $X = \sec t$，则 $dX = \sec t \tan t\,dt$

   代入得：
   $$
   \int \frac{\sec t \tan t}{\sqrt{\sec^2 t - 1}}\,dt = \int \frac{\sec t \tan t}{\tan t}\,dt = \int \sec t\,dt = \ln|\sec t + \tan t| + C
   $$

3. 还原变量：
   - $X = \sec t = x - 1$
   - $\tan t = \sqrt{\sec^2 t - 1} = \sqrt{(x - 1)^2 - 1}$

   所以：
   $$
   \int \frac{dx}{\sqrt{(x - 1)^2 - 1}} = \ln\left| x - 1 + \sqrt{(x - 1)^2 - 1} \right| + C
   $$

---

### ✅ 考试重点 / 易混淆点

- **重点**：掌握三角代换的设定规则；
- **易错点**：
  - 不同形式下 $X$ 和 $t$ 的设定易混淆；
  - 三角函数的代换和化简是常见错点；
  - 正负取值判断错误（如 $\sqrt{X^2 - 1} = \tan t$，但需注意定义域）；
  - 常以计算题形式考查，分值约 10–15 分。

---

## 三、第二类换元法解题步骤（示例二）

### ✅ 解题流程（通用版本）

#### 第一步：设 $X = x + c$，$dX = dx$

- 若 $c = 0$，则 $X = x$，$dX = dx$

#### 第二步：根据根号内形式选择代换

| 根号内形式 | 代换方式 | $dX$ 表达式 |
|-----------|----------|-------------|
| $(x + c)^2 + a^2$ | $X = a \tan t$ | $dX = a \sec^2 t\,dt$ |
| $(x + c)^2 - a^2$ | $X = a \sec t$ | $dX = a \sec t \tan t\,dt$ |
| $a^2 - (x + c)^2$ | $X = a \sin t$ | $dX = a \cos t\,dt$ |

#### 第三步：化简并还原

- 利用三角恒等式（如 $\sin^2 t + \cos^2 t = 1$）化简；
- 积分后将 $t$ 用 $x$ 表示，得到最终答案。

---

### ✅ 示例：计算 $\int \frac{dx}{\sqrt{x^2 + 1}}$

**解**：

1. 设 $X = x$，$dX = dx$，原式变为：
   $$
   \int \frac{dX}{\sqrt{X^2 + 1}}
   $$

2. 含 $\sqrt{X^2 + 1}$，设 $X = \tan t$，则 $dX = \sec^2 t\,dt$

   代入得：
   $$
   \int \frac{\sec^2 t}{\sqrt{\tan^2 t + 1}}\,dt = \int \frac{\sec^2 t}{\sec t}\,dt = \int \sec t\,dt = \ln|\sec t + \tan t| + C
   $$

3. 还原变量：
   - $X = \tan t = x$
   - $\sec t = \sqrt{1 + \tan^2 t} = \sqrt{1 + x^2}$

   所以：
   $$
   \int \frac{dx}{\sqrt{x^2 + 1}} = \ln\left| x + \sqrt{x^2 + 1} \right| + C
   $$

---

### ✅ 考试重点 / 易混淆点

- **重点**：掌握不同形式下的代换技巧；
- **易错点**：
  - 与示例一对比，不同形式下 $X$ 和 $t$ 的设定有差异；
  - 三角函数化简时符号处理不当；
  - 正负取值判断错误（如 $\cos t = \frac{1}{\sqrt{1 + x^2}}$，需注意定义域）；
  - 常以计算题形式考查，分值约 10–15 分。

---

## 四、总结口诀

> **“根号内看结构，配方变标准；  
> 加平方用正切，减平方用正割；  
> 减平方用正弦，三角代换要记牢；  
> 化简积分再还原，答案才完整。”**

---

### 🧠 学习建议

1. **熟练掌握三种标准形式的代换方式**；
2. **多练典型题型**，尤其是含 $\sqrt{x^2 + a^2}$、$\sqrt{x^2 - a^2}$、$\sqrt{a^2 - x^2}$ 的积分；
3. **注意三角函数的恒等变换**，避免计算错误；
4. **重视还原步骤**，确保最终答案以 $x$ 表示。
