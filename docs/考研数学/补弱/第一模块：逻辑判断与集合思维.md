# 考研数学基础深度复习讲义

## 第一部分：逻辑判断与集合思维工具
在考研数学中，解决“充分必要条件”问题的最有效方法不是靠语感，而是靠**集合论**。

### 1. 四种条件的严格定义
设 $A$ 为条件（前件），$B$ 为结论（后件），对应的集合分别为 $S_A$ 和 $S_B$。

* **充分条件** ($A \Rightarrow B$)
    * **含义**：有 $A$ 必有 $B$，“由小推大”。
    * **判别**：集合 $S_A \subseteq S_B$（$A$ 是 $B$ 的子集）。
* **必要条件** ($A \Leftarrow B$)
    * **含义**：要得到 $A$，必须先有 $B$（但有 $B$ 不一定够），“由大推小”。
    * **判别**：集合 $S_A \supseteq S_B$（$A$ 包含 $B$）。
* **充要条件** ($A \Leftrightarrow B$)
    * **含义**：互相可推，两者等价。
    * **判别**：集合 $S_A = S_B$。
* **无关条件**
    * **含义**：两者互不包含，无法推导。

### 2. 核心逻辑运算律
处理复杂命题或进行反证法时的核心工具：

* **逆否命题（等价性）**
    * 原命题 $A \Rightarrow B$ 等价于 $\bar{B} \Rightarrow \bar{A}$。
    * **应用**：当证明“若...则...”很难时，尝试证明“若结论不成立，则条件也不成立”。
* **德·摩根定律（否定律）**
    * **并集的否定**：$\overline{A \cup B} = \bar{A} \cap \bar{B}$（“或者”的否定是“且”）。
    * **交集的否定**：$\overline{A \cap B} = \bar{A} \cup \bar{B}$（“并且”的否定是“或者”）。

## 第二部分：典型逻辑题型深度解析
以下例题展示了如何利用上述“集合思维”解决具体的代数与函数问题。

### 【题型一】数值与不等式关系

**例题 1：代数不等式的包含关系**
> **问题**：判断“$a>1$”是“$a^2 > a$”的什么条件？

* **深度分析**：
    1.  **解析条件 A**：集合为 $(1, +\infty)$。
    2.  **解析条件 B**：解不等式 $a^2 - a > 0 \Rightarrow a(a-1) > 0$。解集为 $(-\infty, 0) \cup (1, +\infty)$。
    3.  **集合比对**：
        * 显然 $(1, +\infty)$ 被包含于 $(-\infty, 0) \cup (1, +\infty)$。即 $A \subsetneq B$。
        * 小范围推大范围 $\Rightarrow$ 充分。
        * 大范围推不回小范围（因为 $B$ 中包含负数部分）$\Rightarrow$ 不必要。
* **结论**：充分非必要条件。

**例题 2：绝对值与高次不等式**
> **问题**：判断“$|x - \frac{1}{2}| < \frac{1}{2}$”是“$x^3 < 1$”的什么条件？

* **深度分析**：
    1.  **解析条件 A**：由 $|x - 0.5| < 0.5$ 得 $-0.5 < x - 0.5 < 0.5$，两边加 0.5 得 $0 < x < 1$。
    2.  **解析条件 B**：由 $x^3 < 1$，因立方函数单调递增，直接开立方得 $x < 1$。
    3.  **集合比对**：
        * 集合 A 是 $(0, 1)$，集合 B 是 $(-\infty, 1)$。
        * 显然 $(0, 1) \subsetneq (-\infty, 1)$。
    4.  **逻辑推导**：$A$ 是 $B$ 的真子集，故 $A \Rightarrow B$，但 $B \nRightarrow A$。
* **结论**：充分非必要条件。

### 【题型二】函数与数列的性质判断

**例题 3：抽象函数的构造与推导（高频难点）**
> **问题**：
> 目标命题 $p$: $\exists a \neq 0, \forall x, f(x+a) < f(x) + f(a)$。
> 条件 $q_1$: $f(x)$ 严格减且 $f(x)>0$。
> 条件 $q_2$: $f(x)$ 严格增且存在 $x_0<0$ 使 $f(x_0)=0$。
> 分析 $q_1, q_2$ 是否为充分条件。

* **深度分析**：
    1.  **分析 $q_1$（充分性成立）**：
        * 我们需要找到**一个** $a$ 使得不等式成立。尝试取 $a > 0$。
        * 因为 $f(x)$ 单调递减，且 $a>0$，所以 $x+a > x$，从而推导出 $f(x+a) < f(x)$。
        * 要让目标不等式 $f(x+a) < f(x) + f(a)$ 成立，我们只需要保证右边多出来的 $f(a)$ 是正数即可。
        * 根据 $q_1$，函数恒大于 0，所以 $f(a) > 0$ 恒成立。
        * 推导链：$f(x+a) < f(x) < f(x) + f(a)$。得证。
    2.  **分析 $q_2$（充分性不成立，举反例）**：
        * $q_2$ 描述的是一个单调增且在负半轴有零点的函数。最简单的模型是直线 $f(x) = x - x_0$（其中 $x_0 < 0$）。
        * 代入目标不等式 $p$ 进行检验：
            * 左边 $= (x+a) - x_0$
            * 右边 $= (x - x_0) + (a - x_0) = x + a - 2x_0$
        * 若不等式成立：$x + a - x_0 < x + a - 2x_0$。
        * 化简得：$-x_0 < -2x_0 \Rightarrow x_0 > 0$。
        * 这与条件“$x_0 < 0$”矛盾。说明对于这类函数，结论不成立。
* **结论**：只有 $q_1$ 是充分条件。

**例题 4：等比数列单调性的陷阱**
> **问题**：设 $\{a_n\}$ 是等比数列，判断“$a_1 < a_2 < a_3$”是“$\{a_n\}$ 严格单调增加”的什么条件？

* **深度分析**：
    1.  **充分性分析（前推后）**：
        * 已知 $a_1 < a_2 < a_3$，即 $a_1 < a_1q < a_1q^2$。
        * **情况一**：若 $a_1 > 0$，则 $1 < q < q^2$，解得 $q > 1$。首项正、公比大于1 $\Rightarrow$ 全局单调增。
        * **情况二**：若 $a_1 < 0$，则 $1 > q > q^2$（变号）。解得 $0 < q < 1$。首项负、公比在0到1之间（例如 -8, -4, -2...） $\Rightarrow$ 数值在数轴上向右移动 $\Rightarrow$ 全局单调增。
        * **综合**：无论 $a_1$ 是正还是负，只要前三项递增，就锁定了公比 $q$ 的范围，使得整个数列必然递增。
    2.  **必要性分析（后推前）**：
        * 若数列全局单调增加，定义即为“任意 $n$ 都有 $a_n < a_{n+1}$”。
        * 既然任意项都满足，那么 $n=1, 2$ 时自然满足 $a_1 < a_2 < a_3$。
* **结论**：充要条件。

## 第三部分：核心证明方法——数学归纳法
这是处理自然数 $n$ 相关命题的标准程序。重点在于**“如何利用假设凑出下一步”**。

### 1. 标准步骤
1.  **奠基**：验证 $n=1$ 时命题成立。
2.  **假设**：假设 $n=k (k \ge 1)$ 时命题成立。
3.  **递推**：**利用 $n=k$ 的结论**，经过代数变形，推导出 $n=k+1$ 时命题也成立。

### 2. 三大经典证明

**证明一：伯努利不等式 (Bernoulli's Inequality)**
> **命题**：若 $x > -1$，则 $(1+x)^n \ge 1+nx$。

* **证明核心**：
    * 在从 $k$ 推导 $k+1$ 时，我们写出左边：$(1+x)^{k+1} = (1+x)^k \cdot (1+x)$。
    * 代入假设：$\ge (1+kx) \cdot (1+x)$。
    * 展开：$= 1 + x + kx + kx^2 = 1 + (k+1)x + kx^2$。
    * **关键一步（放缩）**：因为 $kx^2 \ge 0$，直接去掉这一项，不等式依然成立（即 $\ge 1 + (k+1)x$）。这就是证明的“灵魂”。

**证明二：阶乘与幂的放缩**
> **命题**：$n! < (\frac{n+1}{2})^n$ ($n > 1$)。

* **证明核心（均值不等式视角）**：
    * 此题若用归纳法较为繁琐，通常利用**AM-GM不等式**（算术平均数 $\ge$ 几何平均数）直接秒杀。
    * 考察数列：$1, 2, 3, \dots, n$。
    * 几何平均数：$\sqrt[n]{1 \times 2 \times \cdots \times n} = \sqrt[n]{n!}$。
    * 算术平均数：$\frac{1+2+\cdots+n}{n} = \frac{n(n+1)/2}{n} = \frac{n+1}{2}$。
    * 因为数列各项不相等，故严格小于：$\sqrt[n]{n!} < \frac{n+1}{2}$。两边 $n$ 次方即得证。

**证明三：立方和公式**
> **命题**：$1^3 + 2^3 + \dots + n^3 = [\frac{n(n+1)}{2}]^2$。

* **证明核心**：
    * 计算验证型归纳法。
    * 目标是证明 $n=k+1$ 时，右边等于 $[\frac{(k+1)(k+2)}{2}]^2$。
    * 利用假设：左边 $= [\frac{k(k+1)}{2}]^2 + (k+1)^3$。
    * **代数技巧**：提取公因式 $(k+1)^2$。
    * 剩余部分为 $\frac{k^2}{4} + (k+1) = \frac{k^2+4k+4}{4} = \frac{(k+2)^2}{4}$。
    * 合并后正好凑成目标形式。

## 第四部分：考研数学基础知识体系大纲
根据复习逻辑，建议按以下模块构建知识树：

1.  **基本逻辑模块**
    * 命题判断（充要条件、无关条件）。
    * 逻辑运算（逆否、德摩根）。
    * 证明方法（归纳法、反证法）。

2.  **函数与分析模块**
    * **基本函数**：幂函数、指数函数、对数函数、三角与反三角函数。
    * **核心性质**：单调性、奇偶性（对称问题）、周期性、有界性。
    * **方程**：一元 $n$ 次方程的根与系数关系。

3.  **代数运算模块**
    * **解析式**：有理式拆分、无理式有理化。
    * **不等式**：均值不等式、柯西不等式、绝对值不等式。
    * **数列**：等差/等比数列通项与求和、递推数列的单调性判定。

4.  **几何与坐标模块**
    * 直角坐标系与变换。
    * **极坐标系**：极坐标方程的建立、直角坐标与极坐标的互化（这对后续二重积分计算至关重要）。
