## 题目结构

### Section I: Use of English (完形填空)
- 20道选择题，每题0.5分，共10分

### Section II: Reading Comprehension (阅读理解)

#### Part A
- Text 1: 5道选择题，每题2分，共10分
- Text 2: 5道选择题，每题2分，共10分
- Text 3: 5道选择题，每题2分，共10分
- Text 4: 5道选择题，每题2分，共10分

#### Part B
- 5道选择题，每题2分，共10分

### Section III: Translation (翻译)
- 1道英译中翻译题，共15分

### Section IV: Writing (写作)

#### Part A
- 应用文写作，共10分

#### Part B
- 图表/图画作文，共15分

## 使用建议

1. 先按考试时长完成整套卷，记录得分与卡点。
2. 对照官方或权威解析，标出知识点与易错步骤。
3. 将错题录入错题本，针对薄弱章节回炉练习。
