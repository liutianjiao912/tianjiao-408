## **一、重要极限**  

### 1.1 基本极限（必考核心）

| 公式 | 直觉理解 |
| :--- | :------- |
| $\displaystyle \lim_{x \to 0} \frac{\sin x}{x} = 1$ | 单位圆中弦长≈弧长 |
| $\displaystyle \lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$ | 连续复利模型 |
| $\displaystyle \lim_{x \to 0} (1 + x)^{\frac{1}{x}} = e$ | 指数增长的临界点 |
| $\displaystyle \lim_{x \to 0} \frac{e^x - 1}{x} = 1$ | 指数函数在0点切线斜率 |
| $\displaystyle \lim_{x \to 0} \frac{\ln(1 + x)}{x} = 1$ | 对数函数在0点切线斜率 |

### 1.2 特殊极限（突破难点）

| 公式 | 应用场景 |
| :--- | :------- |
| $\displaystyle \lim_{x \to 0^+} x \ln x = 0$ | 指数-对数转化题型 |
| $\displaystyle \lim_{n \to \infty} \sqrt[n]{n} = 1$ | 数列极限 |
| $\displaystyle \lim_{n \to \infty} \frac{n!}{n^n} = 0$ | 阶乘与幂函数比较 |

## **三、极限与连续运算法则／方法**  

### 3.1 基础法则（前提：极限存在）

- **四则运算**：若 $\displaystyle \lim f=A$, $\displaystyle \lim g=B$，则  
  $\displaystyle \lim(f\pm g)=A\pm B$  
  $\displaystyle \lim(f\cdot g)=A\cdot B$  
  $\displaystyle \lim\frac{f}{g}=\frac{A}{B} \quad (B\neq0)$  
  
- **复合函数极限**：若 $\displaystyle \lim_{x\to x_0}g(x)=u_0$ 且 $\displaystyle \lim_{u\to u_0}f(u)=A$，**且 $g(x)\neq u_0$ 在 $x_0$ 邻域**，则  
  $\displaystyle \lim_{x\to x_0}f(g(x))=A$  
  


### 3.2 核心方法与准则

| 方法 | 适用场景 |
| :--- | :------- |
| **夹逼准则** | 难直接求的振荡/和式极限 |
| **单调有界准则** | 递推数列 $\{x_{n+1}=f(x_n)\}$ |
| **洛必达法则** | $\dfrac{0}{0}$, $\dfrac{\infty}{\infty}$ 型 |
| **泰勒展开** | 0/0型、高精度需求 |

### 3.3 函数连续性

- **定义**：$\displaystyle \lim_{x\to x_0}f(x)=f(x_0)$  
  
- **间断点分类**：  

  | 类型 | 特征 | 实例 |
  | :--- | :--- | :--- |
  | 可去间断 | $\displaystyle \lim_{x\to x_0}f(x)$ 存在 $\neq f(x_0)$ | $f(x)=\dfrac{\sin x}{x}$ 在 $x=0$ |
  | 跳跃间断 | $\displaystyle \lim_{x\to x_0^-}f \neq \lim_{x\to x_0^+}f$ | $f(x)=\lfloor x \rfloor$（取整函数） |
  | 无穷间断 | $\displaystyle \lim_{x\to x_0^+}f(x)=\infty$ 或 $-\infty$，<br>或 $\displaystyle \lim_{x\to x_0^-}f(x)=\infty$ 或 $-\infty$ | $f(x)=\dfrac{1}{x}$ 在 $x=0$ |
  | 振荡间断 | 极限不存在且不趋于 $\infty$ | $f(x)=\sin\dfrac{1}{x}$ 在 $x=0$ |

- **闭区间连续函数性质**（必考证明题基础）：  
  - **有界性定理**：$f\in C[a,b] \Rightarrow \exists M>0$ s.t. $|f(x)|\leq M$  
  - **最值定理**：$f\in C[a,b] \Rightarrow \exists \xi,\eta\in[a,b]$ s.t. $f(\xi)=\max, f(\eta)=\min$  
  - **介值定理**：$f(a)\neq f(b)$ 且 $k\in(f(a),f(b)) \Rightarrow \exists \xi\in(a,b)$ s.t. $f(\xi)=k$  
  - **零点定理**：$f(a)f(b)<0 \Rightarrow \exists \xi\in(a,b)$ s.t. $f(\xi)=0$
