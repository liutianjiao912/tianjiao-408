# 二、知识小结

## 一、无穷小比较题型特点与解题步骤

### ✅ 题型特点

已知当 $x \to x_0$ 时，某含未知数的函数 $\alpha(x)$ 是另一函数 $\beta(x)$ 的某种**无穷小**，求未知参数。

例如：  
> “当 $x \to 0$ 时，$\alpha(x) = 2a f(x) + b f(2x) - f(0)$ 是 $x$ 的高阶无穷小”，求 $a, b$。

这类题目考查对**无穷小阶的判断**和**极限运算能力**。

---

### ✅ 解题步骤（通用流程）

1. **找出两个无穷小函数**：
   - 设 $\alpha(x)$ 为待分析的函数（含未知数）
   - 设 $\beta(x)$ 为已知函数（如 $x$, $x^2$, $\sin x$ 等）

2. **确定极限形式**：
   - 计算：
     $$
     \lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)}
     $$
   - 其中 $x_0$ 是趋于的值（如 $0$ 或 $\infty$）

3. **根据无穷小类型列等式**：
   | 类型 | 极限结果 | 对应关系 |
   |------|----------|----------|
   | **等价无穷小** | $\lim \frac{\alpha(x)}{\beta(x)} = 1$ | $\alpha(x) \sim \beta(x)$ |
   | **同阶无穷小** | $\lim \frac{\alpha(x)}{\beta(x)} = C \ne 0$ | $\alpha(x)$ 与 $\beta(x)$ 同阶 |
   | **高阶无穷小** | $\lim \frac{\alpha(x)}{\beta(x)} = 0$ | $\alpha(x)$ 是 $\beta(x)$ 的高阶无穷小 |
   | **低阶无穷小** | $\lim \frac{\alpha(x)}{\beta(x)} = \infty$ | $\alpha(x)$ 是 $\beta(x)$ 的低阶无穷小 |
   | **k 阶无穷小** | $\lim \frac{\alpha(x)}{\beta(x)^k} = C \ne 0$ | $\alpha(x)$ 是 $\beta(x)$ 的 $k$ 阶无穷小 |

4. **建立方程并求解未知数**

## 二、例题解析与解题示范

### 📌 例题一：高阶无穷小问题

**题目**：设
$$
\alpha(x) = 2a f(x) + b f(2x) - f(0), \quad \beta(x) = x
$$
且当 $x \to 0$ 时，$\alpha(x)$ 是 $x$ 的**高阶无穷小**，求 $a, b$。

**解题步骤**：
1. 找出两个函数：
   - $\alpha(x) = 2a f(x) + b f(2x) - f(0)$
   - $\beta(x) = x$

2. 判断条件：
   - 因为是“高阶无穷小”，所以：
     $$
     \lim_{x \to 0} \frac{\alpha(x)}{x} = 0
     $$

3. 使用洛必达法则或泰勒展开（假设 $f$ 可导）：
   - 若 $f$ 在 $x=0$ 处可导，则：
     $$
     f(x) = f(0) + f'(0)x + o(x), \quad f(2x) = f(0) + 2f'(0)x + o(x)
     $$
   - 代入得：
     $$
     \alpha(x) = 2a[f(0) + f'(0)x] + b[f(0) + 2f'(0)x] - f(0) + o(x)
     $$
     $$
     = (2a f(0) + b f(0) - f(0)) + (2a f'(0) + 2b f'(0))x + o(x)
     $$
     $$
     = f(0)(2a + b - 1) + 2f'(0)(a + b)x + o(x)
     $$

4. 要使 $\alpha(x)$ 是 $x$ 的高阶无穷小，必须满足：
   - 常数项为 0：$2a + b - 1 = 0$
   - 一次项也为 0：$a + b = 0$

5. 解方程组：
   $$
   \begin{cases}
   2a + b = 1 \\
   a + b = 0
   \end{cases}
   \Rightarrow a = 1, b = -1
   $$

✅ 答案：$a = 1$, $b = -1$

### 📌 例题二：等价无穷小问题

**题目**：设
$$
\alpha(x) = \cos(ax) - 1 + \sin(ax), \quad \beta(x) = 3x
$$
且当 $x \to 0$ 时，$\alpha(x)$ 与 $\beta(x)$ 是**等价无穷小**，求 $a$。

**解题步骤**：
1. 等价无穷小意味着：
   $$
   \lim_{x \to 0} \frac{\alpha(x)}{\beta(x)} = 1
   $$

2. 使用泰勒展开（$x \to 0$）：
   - $\cos(ax) = 1 - \frac{(ax)^2}{2} + o(x^2)$
   - $\sin(ax) = ax - \frac{(ax)^3}{6} + o(x^3)$
   - 所以：
     $$
     \alpha(x) = \left(1 - \frac{a^2 x^2}{2}\right) - 1 + \left(ax - \frac{a^3 x^3}{6}\right) + o(x^3)
     = ax - \frac{a^2 x^2}{2} + o(x^2)
     $$

3. 分母 $\beta(x) = 3x$

4. 求极限：
   $$
   \frac{\alpha(x)}{\beta(x)} = \frac{ax - \frac{a^2 x^2}{2} + o(x^2)}{3x} = \frac{a}{3} - \frac{a^2 x}{6} + o(x)
   $$
   当 $x \to 0$ 时，极限为 $\frac{a}{3}$

5. 令其等于 1：
   $$
   \frac{a}{3} = 1 \Rightarrow a = 3
   $$

✅ 答案：$a = 3$

## 三、关键概念定义与辨析

### ✅ 高阶无穷小

- **定义**：若 $\lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)} = 0$，则称 $\alpha(x)$ 是 $\beta(x)$ 的**高阶无穷小**。
- **含义**：$\alpha(x)$ 比 $\beta(x)$ 更快趋于 0。
- **例子**：$\lim_{x \to 0} \frac{x^2}{x} = 0$，所以 $x^2$ 是 $x$ 的高阶无穷小。

---

### ✅ 低阶无穷小

- **定义**：若 $\lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)} = \infty$，则称 $\alpha(x)$ 是 $\beta(x)$ 的**低阶无穷小**。
- **含义**：$\alpha(x)$ 比 $\beta(x)$ 更慢趋于 0。
- **例子**：$\lim_{x \to 0} \frac{x}{x^2} = \infty$，所以 $x$ 是 $x^2$ 的低阶无穷小。

---

### ✅ 同阶无穷小

- **定义**：若 $\lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)} = C \ne 0$，则称 $\alpha(x)$ 与 $\beta(x)$ 是**同阶无穷小**。
- **性质**：两者趋于 0 的速度相同。
- **例子**：$\lim_{x \to 0} \frac{\frac{1}{2}x}{x} = \frac{1}{2}$，所以 $\frac{1}{2}x$ 与 $x$ 是同阶无穷小。

---

### ✅ 等价无穷小

- **定义**：若 $\lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)} = 1$，则称 $\alpha(x) \sim \beta(x)$，即**等价无穷小**。
- **性质**：在极限计算中可以相互替换。
- **常见等价关系**（$x \to 0$）：
  - $\sin x \sim x$
  - $\tan x \sim x$
  - $1 - \cos x \sim \frac{x^2}{2}$
  - $\ln(1 + x) \sim x$
  - $e^x - 1 \sim x$

---

### ✅ k 阶无穷小

- **定义**：若 $\lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)^k} = C \ne 0$，则称 $\alpha(x)$ 是 $\beta(x)$ 的**k 阶无穷小**。
- **含义**：$\alpha(x)$ 与 $\beta(x)^k$ 是同阶无穷小。
- **例子**：$\lim_{x \to 0} \frac{\frac{1}{2}x^3}{x^3} = \frac{1}{2}$，所以 $\frac{1}{2}x^3$ 是 $x$ 的**三阶无穷小**。

### ❌ 易混淆点

| 问题 | 说明 |
|------|------|
| **混淆等价与同阶** | 等价要求极限为 1，同阶只要求非零常数。 |
| **忽略隐藏信息** | 如“某式是另一式的高阶无穷小”意味着极限为 0，需建立方程。 |
| **误用洛必达法则** | 必须满足 $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$ 形式，否则不能使用。 |
| **忽略阶数匹配** | 如 $x^2$ 是 $x$ 的高阶无穷小，但不是 $x^3$ 的高阶无穷小。 |
