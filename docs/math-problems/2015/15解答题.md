## 🎯 问题回顾

设函数：
$$f(x) = x + a\ln(1+x) + bx\sin x$$
$$g(x) = kx^3$$

若 $f(x)$ 与 $g(x)$ 在 $x \to 0$ 时是**等价无穷小**，求常数 $a, b, k$ 的值。

---

## 第一部分：等价无穷小的定义与分析

### 1.1 定义回顾

若 $\alpha(x)$ 与 $\beta(x)$ 在 $x \to 0$ 时是等价无穷小（记作 $\alpha(x) \sim \beta(x)$），则满足：
$$\lim_{x \to 0} \frac{\alpha(x)}{\beta(x)} = 1$$

在本题中，意味着：
$$\lim_{x \to 0} \frac{f(x)}{kx^3} = 1$$

### 1.2 解题策略分析

由于分母 $g(x) = kx^3$ 是关于 $x$ 的 **3 阶无穷小**，为了使极限存在且为 1，分子 $f(x)$ 的泰勒展开式中：
1.  **常数项、一次项 ($x$)、二次项 ($x^2$) 的系数必须为 0**（否则极限为 $\infty$）。
2.  **三次项 ($x^3$) 的系数必须等于 $k$**（使比值极限为 1）。

因此，我们需要将 $f(x)$ 在 $x=0$ 处进行**泰勒展开（麦克劳林展开）**至 $x^3$ 项。

---

## 第二部分：泰勒展开计算

### 2.1 常用展开公式

我们需要用到以下两个标准展开式（展开到 $o(x^3)$）：

1.  **对数函数**：
    $$\ln(1+x) = x - \frac{1}{2}x^2 + \frac{1}{3}x^3 + o(x^3)$$

2.  **正弦函数**（注意需乘以 $bx$）：
    $$\sin x = x - \frac{1}{6}x^3 + o(x^3)$$
    $$\implies bx\sin x = bx\left(x + o(x^2)\right) = bx^2 + o(x^3)$$

### 2.2 代入 $f(x)$ 并整理

将上述展开式代入 $f(x)$ 的表达式：

$$
\begin{aligned}
f(x) &= x + a\ln(1+x) + bx\sin x \\
&= x + a\left( x - \frac{1}{2}x^2 + \frac{1}{3}x^3 + o(x^3) \right) + \left( bx^2 + o(x^3) \right)
\end{aligned}
$$

### 2.3 按幂次合并同类项

$$
f(x) = (1+a)x + \left( b - \frac{a}{2} \right)x^2 + \frac{a}{3}x^3 + o(x^3)
$$

---

## 第三部分：系数比较与方程求解

### 3.1 建立方程组

根据等价无穷小的条件 $f(x) \sim kx^3$，分子中低于 3 次的项系数必须为 0，且 3 次项系数必须等于 $k$。

由此得到以下方程组：

1.  **$x$ 的系数为 0**：
    $$1 + a = 0$$

2.  **$x^2$ 的系数为 0**：
    $$b - \frac{a}{2} = 0$$

3.  **$x^3$ 的系数等于 $k$**：
    $$\frac{a}{3} = k$$

### 3.2 求解方程组

1.  由方程 ①：
    $$a = -1$$

2.  将 $a = -1$ 代入方程 ②：
    $$b - \frac{-1}{2} = 0 \implies b + \frac{1}{2} = 0 \implies b = -\frac{1}{2}$$

3.  将 $a = -1$ 代入方程 ③：
    $$k = \frac{-1}{3} = -\frac{1}{3}$$

---

## ✅ 最终答案

$$
a = -1, \quad b = -\frac{1}{2}, \quad k = -\frac{1}{3}
$$

---

## 📌 关键要点总结

1.  **识别阶数**：看到“等价无穷小”且一方为 $kx^n$，应立即想到将另一方展开至 $x^n$，并令低阶项系数为 0，同阶项系数相等。
2.  **泰勒公式的熟练度**：$\ln(1+x)$ 和 $\sin x$ 是考研数学中最核心的展开式，务必熟记系数符号（$\ln$ 是正负交替，$\sin$ 是奇数阶乘）。
3.  **运算细节**：在处理 $bx\sin x$ 时，因外部已有 $x$，内部展开只需到 $x^2$ 即可得到整体的 $x^3$ 精度，避免过度展开浪费时间。
