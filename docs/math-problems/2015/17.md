### **题目：**

> (17)（本题满分 11 分）
> 已知函数 $f(x,y)$ 满足：
> - $f_{xy}(x,y) = 2(y+1)e^x$
> - $f_x(x,0) = (x+1)e^x$
> - $f(0,y) = y^2 + 2y$
> 求 $f(x,y)$ 的极值。

> 【答案】极小值 $f(0,-1) = -1$

> 【解析】
> 对 $f_{xy}(x,y) = 2(y+1)e^x$ 两边对 $y$ 积分，得：
> $$
> f_x(x,y) = 2\left(\frac{1}{2}y^2 + y\right)e^x + \varphi(x) = (y^2 + 2y)e^x + \varphi(x)
> $$
> 由 $f_x(x,0) = (x+1)e^x$，得 $\varphi(x) = (x+1)e^x$
>
> 所以：
> $$
> f_x(x,y) = (y^2 + 2y)e^x + (x+1)e^x
> $$
> 对 $x$ 积分得：
> $$
> f(x,y) = (y^2 + 2y)e^x + \int (x+1)e^x dx
> $$
> 计算积分：
> $$
> \int (x+1)e^x dx = (x+1)e^x - \int e^x dx = (x+1)e^x - e^x = x e^x + C
> $$
> 所以：
> $$
> f(x,y) = (y^2 + 2y)e^x + x e^x + C
> $$
> 由 $f(0,y) = y^2 + 2y$，代入 $x=0$ 得：
> $$
> f(0,y) = (y^2 + 2y)\cdot 1 + 0 + C = y^2 + 2y + C
> $$
> 与已知 $f(0,y) = y^2 + 2y$ 比较，得 $C = 0$
>
> 故：
> $$
> f(x,y) = (y^2 + 2y)e^x + x e^x
> $$
>
> 令偏导数为零：
> $$
> \begin{cases}
> f_x = (y^2 + 2y)e^x + e^x + x e^x = 0 \\
> f_y = (2y + 2)e^x = 0
> \end{cases}
> $$
> 解得 $y = -1$，代入第一式得 $x = 0$
>
> 二阶导数：
> $$
> f_{xx} = (y^2 + 2y)e^x + 2e^x + x e^x, \quad f_{xy} = 2(y+1)e^x, \quad f_{yy} = 2e^x
> $$
> 在 $(0,-1)$ 处：
> - $A = f_{xx}(0,-1) = (1 - 2)e^0 + 2e^0 + 0 = (-1) + 2 = 1$
> - $B = f_{xy}(0,-1) = 2(0)e^0 = 0$
> - $C = f_{yy}(0,-1) = 2e^0 = 2$
> - $AC - B^2 = 1 \cdot 2 - 0 = 2 > 0$，且 $A > 0$，故为极小值
>
> $f(0,-1) = (1 - 2)\cdot 1 + 0 = -1$

---

我们将从以下几个方面进行系统分析：

---

## ✅ 一、题意理解

给定一个二元函数 $f(x,y)$，但未显式给出，而是通过其**混合偏导数**和**边界条件**来确定。

要求：
- 先求出 $f(x,y)$ 的表达式；
- 再求其极值点及极值。

这是一个典型的"**由高阶偏导数反推原函数**"的问题，属于多元函数微分学中的综合题。

---

## ✅ 二、核心思想：**逐步积分法**

由于已知的是二阶混合偏导数 $f_{xy}$，我们可以：
1. 先对 $y$ 积分，得到 $f_x(x,y)$
2. 再对 $x$ 积分，得到 $f(x,y)$
3. 利用初始条件确定常数项

这是解决此类问题的标准方法。

---

## ✅ 三、第一步：对 $f_{xy}(x,y)$ 关于 $y$ 积分

已知：

$$
f_{xy}(x,y) = 2(y+1)e^x
$$

注意：$f_{xy} = \frac{\partial}{\partial y} \left( f_x(x,y) \right)$

所以对 $y$ 积分，得到 $f_x(x,y)$：

$$
f_x(x,y) = \int 2(y+1)e^x dy = 2e^x \int (y+1) dy = 2e^x \left( \frac{1}{2}y^2 + y \right) + \varphi(x)
= (y^2 + 2y)e^x + \varphi(x)
$$

其中 $\varphi(x)$ 是关于 $x$ 的任意函数（因为对 $y$ 积分时，$x$ 视为常数）。

---

## ✅ 四、利用条件 $f_x(x,0) = (x+1)e^x$ 确定 $\varphi(x)$

将 $y = 0$ 代入上式：

$$
f_x(x,0) = (0 + 0)e^x + \varphi(x) = \varphi(x)
$$

但题目给出 $f_x(x,0) = (x+1)e^x$，所以：

$$
\varphi(x) = (x+1)e^x
$$

因此：

$$
f_x(x,y) = (y^2 + 2y)e^x + (x+1)e^x
$$

---

## ✅ 五、第二步：对 $f_x(x,y)$ 关于 $x$ 积分，求 $f(x,y)$

$$
f(x,y) = \int f_x(x,y) dx = \int \left[ (y^2 + 2y)e^x + (x+1)e^x \right] dx
$$

注意到 $y$ 是常数，可以提出：

$$
= (y^2 + 2y) \int e^x dx + \int (x+1)e^x dx
$$

第一个积分简单：

$$
\int e^x dx = e^x
$$

第二个积分用分部积分：

令 $u = x+1$, $dv = e^x dx$ → $du = dx$, $v = e^x$

$$
\int (x+1)e^x dx = (x+1)e^x - \int e^x dx = (x+1)e^x - e^x = x e^x
$$

所以：

$$
f(x,y) = (y^2 + 2y)e^x + x e^x + C
$$

其中 $C$ 是常数（因为对 $x$ 积分，$y$ 无关）。

---

## ✅ 六、利用条件 $f(0,y) = y^2 + 2y$ 确定 $C$

将 $x = 0$ 代入：

$$
f(0,y) = (y^2 + 2y)e^0 + 0 \cdot e^0 + C = y^2 + 2y + C
$$

但题目给出 $f(0,y) = y^2 + 2y$，所以：

$$
y^2 + 2y + C = y^2 + 2y \Rightarrow C = 0
$$

✅ 所以最终函数为：

$$
\boxed{f(x,y) = (y^2 + 2y)e^x + x e^x}
$$

---

## ✅ 七、求极值：先找驻点

令偏导数为零。

### 1. 计算 $f_x$ 和 $f_y$

#### $f_x$：

$$
f_x = \frac{\partial}{\partial x} \left[ (y^2 + 2y)e^x + x e^x \right] = (y^2 + 2y)e^x + e^x + x e^x = (y^2 + 2y + 1 + x)e^x
$$

即：

$$
f_x = (y^2 + 2y + 1 + x)e^x = ((y+1)^2 + x)e^x
$$

#### $f_y$：

$$
f_y = \frac{\partial}{\partial y} \left[ (y^2 + 2y)e^x + x e^x \right] = (2y + 2)e^x = 2(y+1)e^x
$$

---

### 2. 联立方程求驻点

令：
$$
\begin{cases}
f_x = 0 \\
f_y = 0
\end{cases}
\Rightarrow
\begin{cases}
((y+1)^2 + x)e^x = 0 \\
2(y+1)e^x = 0
\end{cases}
$$

注意：$e^x > 0$ 恒成立，所以可以除掉。

从第二式：

$$
2(y+1)e^x = 0 \Rightarrow y+1 = 0 \Rightarrow y = -1
$$

代入第一式：

$$
(( -1 + 1 )^2 + x)e^x = (0 + x)e^x = x e^x = 0 \Rightarrow x = 0
$$

所以唯一驻点为 $(0, -1)$

---

## ✅ 八、判断极值类型：二阶导数判别法

计算二阶偏导数：

### 1. $f_{xx}$

$$
f_{xx} = \frac{\partial}{\partial x} \left[ ((y+1)^2 + x)e^x \right] = e^x + ((y+1)^2 + x)e^x = (y^2 + 2y + 1 + x + 1)e^x
$$

更简洁地：

$$
f_{xx} = (y^2 + 2y + 1 + x + 1)e^x = (y^2 + 2y + x + 2)e^x
$$

或直接从前面表达式：

$$
f_x = (y^2 + 2y + 1 + x)e^x
\Rightarrow f_{xx} = (y^2 + 2y + 1 + x)' \cdot e^x + (y^2 + 2y + 1 + x) \cdot e^x = e^x + (y^2 + 2y + 1 + x)e^x
= (y^2 + 2y + x + 2)e^x
$$

### 2. $f_{xy}$

$$
f_{xy} = \frac{\partial}{\partial y} \left[ (y^2 + 2y + 1 + x)e^x \right] = (2y + 2)e^x = 2(y+1)e^x
$$

### 3. $f_{yy}$

$$
f_{yy} = \frac{\partial}{\partial y} \left[ 2(y+1)e^x \right] = 2e^x
$$

---

### 4. 在驻点 $(0, -1)$ 处计算：

- $A = f_{xx}(0,-1) = ((-1)^2 + 2(-1) + 0 + 2)e^0 = (1 - 2 + 2) \cdot 1 = 1$
- $B = f_{xy}(0,-1) = 2(-1 + 1)e^0 = 0$
- $C = f_{yy}(0,-1) = 2e^0 = 2$

判别式：

$$
AC - B^2 = 1 \cdot 2 - 0^2 = 2 > 0
$$

且 $A = 1 > 0$，所以该点为**极小值点**

极小值为：
$$
f(0,-1) = ((-1)^2 + 2(-1))e^0 + 0 = (1 - 2) \cdot 1 = -1
$$

---

## ✅ 九、结论

- 函数表达式为：$f(x,y) = (y^2 + 2y)e^x + x e^x$
- 唯一驻点：$(0, -1)$
- 该点为极小值点，极小值为 $f(0,-1) = -1$

---

## ✅ 十、关键点总结

| 步骤 | 内容 |
|------|------|
| ① | 从 $f_{xy}$ 开始，先对 $y$ 积分得 $f_x$ |
| ② | 利用 $f_x(x,0)$ 确定积分常数函数 $\varphi(x)$ |
| ③ | 对 $f_x$ 对 $x$ 积分得 $f(x,y)$ |
| ④ | 利用 $f(0,y)$ 确定常数 $C$ |
| ⑤ | 求偏导，找驻点 |
| ⑥ | 用二阶导数判别法判断极值类型 |

---

## ✅ 十一、常见易错点提醒

| 错误类型 | 说明 |
|--------|------|
| ❌ 积分时漏掉函数项 | 如 $f_x = (y^2 + 2y)e^x + \varphi(x)$，不能写成 $(y^2 + 2y)e^x + c$ |
| ❌ 分部积分错误 | $\int (x+1)e^x dx$ 应为 $(x+1)e^x - e^x = x e^x$ |
| ❌ 忽略 $e^x$ 恒正 | 不能直接说 $f_x = 0$ 当且仅当括号为 0 |
| ❌ 驻点判断不完整 | 必须验证是否为极值，不能只看驻点 |
| ❌ 二阶导数计算错误 | 特别是 $f_{xy}$ 不等于 $f_{yx}$？但在连续情况下相等 |

---

## ✅ 十二、结论

本题考查了**多元函数的构造与极值求解**，涉及：
- 二阶偏导数的积分还原
- 初始条件的应用
- 极值的判定

是一道综合性较强的题目。

$$
\boxed{\text{极小值 } f(0,-1) = -1}
$$
