这道题是**由隐式方程确定的二元函数极值问题**的典型代表，综合考查了**隐函数求偏导、驻点求解、二阶导数判别法（Hessian 判别）** 等核心知识点。我们来对题目进行**系统、严谨且深入的解析**。

---

## 一、题目回顾

已知函数 $ z = z(x, y) $ 由方程  
$$
(x^2 + y^2)z + \ln z + 2(x + y + 1) = 0 \tag{*}
$$  
所确定，求其极值。

**答案：** 极大值为 $ z(-1, -1) = 1 $

---

## 二、解题思路总览

对于由方程 $ F(x, y, z) = 0 $ 隐式定义的函数 $ z = z(x, y) $，其极值点满足：

1. **一阶必要条件**：$ \frac{\partial z}{\partial x} = 0 $，$ \frac{\partial z}{\partial y} = 0 $
2. **代入原方程与导数方程**，解出驻点 $ (x_0, y_0, z_0) $
3. **计算二阶偏导数** $ A = z_{xx}, B = z_{xy}, C = z_{yy} $
4. **利用判别式** $ AC - B^2 $ 判断极值类型[ty-reference](1)：
   - 若 $ AC - B^2 > 0 $ 且 $ A < 0 $ ⇒ 极大值
   - 若 $ AC - B^2 > 0 $ 且 $ A > 0 $ ⇒ 极小值
   - 若 $ AC - B^2 < 0 $ ⇒ 鞍点
   - 若 $ AC - B^2 = 0 $ ⇒ 不定

---

## 三、详细推导

### 第一步：设隐函数方程

令：
$$
F(x, y, z) = (x^2 + y^2)z + \ln z + 2(x + y + 1)
$$
则 $ F(x, y, z) = 0 $ 定义了 $ z = z(x, y) $（在 $ z > 0 $ 区域内，因 $ \ln z $ 要求 $ z > 0 $）。

---

### 第二步：求一阶偏导（隐函数求导）

对 (*) 式两边关于 $ x $ 求偏导（注意 $ z = z(x, y) $）：

$$
\frac{\partial}{\partial x} \left[ (x^2 + y^2)z + \ln z + 2(x + y + 1) \right] = 0
$$

应用乘积法则和链式法则：

$$
2x z + (x^2 + y^2) z_x + \frac{1}{z} z_x + 2 = 0 \tag{1}
$$

同理，对 $ y $ 求偏导：

$$
2y z + (x^2 + y^2) z_y + \frac{1}{z} z_y + 2 = 0 \tag{2}
$$

其中 $ z_x = \frac{\partial z}{\partial x} $，$ z_y = \frac{\partial z}{\partial y} $

---

### 第三步：求驻点（令一阶偏导为零）

设极值点处 $ z_x = 0 $，$ z_y = 0 $，代入 (1)(2) 得：

$$
2x z + 2 = 0 \quad \Rightarrow \quad x = -\frac{1}{z} \\
2y z + 2 = 0 \quad \Rightarrow \quad y = -\frac{1}{z}
$$

所以：
$$
x = y = -\frac{1}{z} \tag{3}
$$

将 (3) 代入原方程 (*)：

- $ x^2 + y^2 = \frac{1}{z^2} + \frac{1}{z^2} = \frac{2}{z^2} $
- $ x + y + 1 = -\frac{1}{z} - \frac{1}{z} + 1 = 1 - \frac{2}{z} $

代入得：
$$
\left( \frac{2}{z^2} \right) z + \ln z + 2\left(1 - \frac{2}{z}\right) = 0 \\
\Rightarrow \frac{2}{z} + \ln z + 2 - \frac{4}{z} = 0 \\
\Rightarrow \ln z - \frac{2}{z} + 2 = 0 \tag{4}
$$

令 $ g(z) = \ln z - \frac{2}{z} + 2 $，定义域 $ z > 0 $

求导：
$$
g'(z) = \frac{1}{z} + \frac{2}{z^2} > 0 \quad (\forall z > 0)
$$

⇒ $ g(z) $ 严格单调递增

又 $ g(1) = 0 - 2 + 2 = 0 $，故 **唯一解为 $ z = 1 $**

代入 (3) 得：
$$
x = y = -1
$$

✅ 唯一驻点为 $ (-1, -1) $，对应 $ z = 1 $

---

### 第四步：求二阶偏导，判断极值类型

我们需要计算在 $ (-1, -1, 1) $ 处的：
- $ A = z_{xx} $
- $ B = z_{xy} $
- $ C = z_{yy} $

#### 1. 求 $ A = z_{xx} $

对 (1) 式两边再对 $ x $ 求偏导：

左边逐项求导：
- $ \frac{\partial}{\partial x}(2x z) = 2z + 2x z_x $
- $ \frac{\partial}{\partial x}[(x^2 + y^2) z_x] = 2x z_x + (x^2 + y^2) z_{xx} $
- $ \frac{\partial}{\partial x}\left( \frac{1}{z} z_x \right) = -\frac{1}{z^2} z_x^2 + \frac{1}{z} z_{xx} $
- $ \frac{\partial}{\partial x}(2) = 0 $

合并得：
$$
2z + 2x z_x + 2x z_x + (x^2 + y^2) z_{xx} - \frac{1}{z^2} z_x^2 + \frac{1}{z} z_{xx} = 0
$$

在驻点 $ (-1, -1, 1) $ 处：
- $ z = 1 $
- $ z_x = 0 $
- $ x^2 + y^2 = 1 + 1 = 2 $

代入：
$$
2(1) + 0 + 0 + 2A - 0 + 1 \cdot A = 0 \Rightarrow 2 + 3A = 0 \Rightarrow A = -\frac{2}{3}
$$

#### 2. 求 $ C = z_{yy} $

由对称性（方程关于 $ x, y $ 对称），可得：
$$
C = z_{yy} = A = -\frac{2}{3}
$$

#### 3. 求 $ B = z_{xy} $

对 (1) 式两边对 $ y $ 求偏导：

- $ \frac{\partial}{\partial y}(2x z) = 2x z_y $
- $ \frac{\partial}{\partial y}[(x^2 + y^2) z_x] = 2y z_x + (x^2 + y^2) z_{xy} $
- $ \frac{\partial}{\partial y}\left( \frac{1}{z} z_x \right) = -\frac{1}{z^2} z_y z_x + \frac{1}{z} z_{xy} $
- 其他项导数为 0

在驻点处 $ z_x = z_y = 0 $，所以前两项为 0，交叉项也为 0

得：
$$
(x^2 + y^2) B + \frac{1}{z} B = 0 \Rightarrow (2 + 1) B = 0 \Rightarrow B = 0
$$

---

### 第五步：判别极值

计算判别式：
$$
AC - B^2 = \left( -\frac{2}{3} \right)\left( -\frac{2}{3} \right) - 0 = \frac{4}{9} > 0
$$

且 $ A = -\frac{2}{3} < 0 $

✅ 故 $ z(-1, -1) = 1 $ 为**极大值**

---

## 四、关键细节与注意事项

| 要点 | 说明 |
|------|------|
| **定义域** | $ \ln z $ 要求 $ z > 0 $，所有计算在此前提下进行 |
| **唯一驻点** | 通过单调性证明 $ z = 1 $ 是唯一解，避免多解混淆 |
| **对称性利用** | 节省计算量，$ z_{yy} = z_{xx} $ |
| **二阶导计算** | 注意乘积法则和链式法则，尤其 $ \frac{1}{z} z_x $ 的导数 |

---

## 五、总结

本题通过以下步骤完成：

1. **隐函数求一阶偏导**，建立极值必要条件；
2. **联立求解驻点**，利用单调性确定唯一解；
3. **求二阶偏导**，利用对称性和代入简化；
4. **Hessian 判别法**判断极值类型。

✅ **最终结论**：
函数 $ z = z(x, y) $ 在点 $ (-1, -1) $ 处取得**极大值**，极大值为：
$$
\boxed{1}
$$
