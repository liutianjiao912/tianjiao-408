title: 2016 数学二真题 第5题
---

这道题综合考查了**曲率、凹凸性、切线与函数局部比较**的关系，是一道典型的微分几何与分析结合的题目。我们来对题目进行**系统、严谨且直观的详细解析**。

---

## 一、题设条件梳理

给定：
- 函数 $ f_1(x), f_2(x) $ 在点 $ x_0 $ 处具有 **二阶连续导数**；
- $ f_1''(x_0) < 0 $，$ f_2''(x_0) < 0 $ ⇒ 两曲线在 $ x_0 $ 附近都是**上凸（或称“下凹”）** 的；
- 两曲线在点 $ (x_0, y_0) $ 处有**公切线** $ y = g(x) $，即：
  - $ f_1(x_0) = f_2(x_0) = g(x_0) = y_0 $
  - $ f_1'(x_0) = f_2'(x_0) = g'(x_0) $
- 曲率满足：$ K_1 > K_2 $，其中 $ K_i $ 是 $ y = f_i(x) $ 在 $ x_0 $ 处的曲率。

问：在 $ x_0 $ 的某个邻域内，三者大小关系如何？

---

## 二、关键知识点回顾

### 1. **曲率公式（平面曲线 $ y = f(x) $）**

$$
K = \frac{|f''(x)|}{\left(1 + (f'(x))^2\right)^{3/2}}
$$

由于在 $ x_0 $ 处两曲线有相同的一阶导数（公切线），即 $ f_1'(x_0) = f_2'(x_0) $，所以分母相同。

因此，
$$
K_1 > K_2 \quad \Longleftrightarrow \quad |f_1''(x_0)| > |f_2''(x_0)|
$$

又因 $ f_i''(x_0) < 0 $，故绝对值越大，实际值越小（更负）：

$$
f_1''(x_0) < f_2''(x_0) < 0
$$

✅ 这是核心不等式。

---

### 2. **凹凸性与切线位置关系**

- 若 $ f''(x_0) < 0 $，则函数在该点附近是**上凸**（开口向下）；
- 对于上凸函数，其图像**位于任意切线的下方**，即：
  $$
  f(x) \leq g(x) \quad \text{在 } x_0 \text{ 附近}
  $$
  其中 $ g(x) $ 是在 $ x_0 $ 处的切线。

因此，对 $ i=1,2 $，都有：
$$
f_i(x) \leq g(x)
$$

---

### 3. **比较 $ f_1(x) $ 与 $ f_2(x) $ 的大小**

构造差函数：
$$
F(x) = f_1(x) - f_2(x)
$$

由题设：
- $ F(x_0) = 0 $
- $ F'(x_0) = f_1'(x_0) - f_2'(x_0) = 0 $
- $ F''(x_0) = f_1''(x_0) - f_2''(x_0) < 0 $

⇒ 根据**极值第二充分条件**，$ x_0 $ 是 $ F(x) $ 的**极大值点**。

因此，在 $ x_0 $ 的某个邻域内：
$$
F(x) \leq F(x_0) = 0 \quad \Rightarrow \quad f_1(x) \leq f_2(x)
$$

---

## 三、综合结论

我们已得：
- $ f_1(x) \leq f_2(x) $
- $ f_2(x) \leq g(x) $

⇒ 联立得：
$$
f_1(x) \leq f_2(x) \leq g(x)
$$

对应选项 **A**。

---

## 四、几何直观解释（法二：特殊值法）

取两个**同心圆弧**（更准确地说，是不同半径、同一切点的圆弧）：

- 小圆（半径 $ R_1 = 2 $）：曲率 $ K_1 = 1/2 $
- 大圆（半径 $ R_2 = 3 $）：曲率 $ K_2 = 1/3 $

在最低点 $ (0, 0) $ 处，两者都与 $ x $-轴相切（公切线 $ y = 0 $），且都是上凸曲线。

由于小圆“弯得更厉害”，它在切点附近**更贴近切线下方**，即：
$$
f_1(x) \leq f_2(x) \leq 0 = g(x)
$$

图形如下（想象）：

```
       ● (0,3)           ● (0,2)
        \                 /
         \               /
          \             /
           \___________/   ← 切线 y=0
                (0,0)
```

小圆（内侧）更低 ⇒ $ f_1(x) \leq f_2(x) $

✅ 直观验证选项 A 正确。

---

## 五、常见误区辨析

| 误区 | 正确认知 |
|------|----------|
| “曲率大 = 函数值大” | 错！曲率描述**弯曲程度**，不是高低。上凸时，曲率越大，图像越“贴近”切线下方 |
| 忽略 $ f'' < 0 $ 的符号 | 必须利用符号判断谁更“凹”（即谁的二阶导更负） |
| 混淆凹凸方向 | $ f'' < 0 $ ⇒ 上凸 ⇒ 图像在切线下方 |

---

## 六、总结

本题通过**曲率大小 → 二阶导大小 → 差函数极值 → 函数局部大小关系**，层层递进，考察了：

- 曲率公式的应用
- 凹凸性与切线位置关系
- 泰勒展开思想（隐含在极值判定中）
- 几何直观与代数推理结合

**最终结论：**

> 在 $ x_0 $ 附近，曲率更大的上凸曲线更“贴近”切线下方，因此：
> $$
> f_1(x) \leq f_2(x) \leq g(x)
> $$

✅ **答案：A**
