3. 设数列 $\{x_{n}\}$ 收敛，则 ( )

   (A) 若 $\lim_{n\rightarrow\infty}\sin x_{n}=0$，则 $\lim_{n\rightarrow\infty}x_{n}=0$

   (B) 若 $\lim_{n\rightarrow\infty}(x_{n}+\sqrt{|x_{n}|})=0$，则 $\lim_{n\rightarrow\infty}x_{n}=0$

   (C) 若 $\lim_{n\rightarrow\infty}(x_{n}+x_{n}^2)=0$，则 $\lim_{n\rightarrow\infty}x_{n}=0$

   (D) 若 $\lim_{n\rightarrow\infty}(x_{n}+\sin x_{n})=0$，则 $\lim_{n\rightarrow\infty}x_{n}=0$
