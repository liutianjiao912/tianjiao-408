## 🔍 一、几何条件转化为代数方程

设曲线 $ y = y(x) $ 在点 $ P(x, y) $ 处：

- **切线方程**（点斜式）：
  $$
  Y - y = y'(X - x)
  $$
  令 $ X = 0 $（与 $ y $ 轴交点），得：
  $$
  Y_P = y - x y'
  $$

- **法线方程**（斜率为 $ -1/y' $）：
  $$
  Y - y = -\frac{1}{y'}(X - x)
  $$
  令 $ Y = 0 $（与 $ x $ 轴交点），得：
  $$
  0 - y = -\frac{1}{y'}(X_P - x) \Rightarrow X_P = x + y y'
  $$

题目条件：**$ X_P = Y_P $**，即：
$$
x + y y' = y - x y'
$$

整理：
$$
x - y + y y' + x y' = 0 \Rightarrow (x - y) + y'(x + y) = 0
$$

解出导数：
$$
y' = \frac{dy}{dx} = \frac{y - x}{x + y}
$$

✅ 这是一个**齐次微分方程**（分子分母均为一次齐次式）。

## 🧮 二、变量代换求解齐次方程

令 $ u = \frac{y}{x} \Rightarrow y = ux $，则：
$$
y' = u + x \frac{du}{dx}
$$

代入方程：
$$
u + x u' = \frac{ux - x}{x + ux} = \frac{x(u - 1)}{x(1 + u)} = \frac{u - 1}{1 + u}
$$

移项：
$$
x u' = \frac{u - 1}{1 + u} - u = \frac{u - 1 - u(1 + u)}{1 + u} = \frac{u - 1 - u - u^2}{1 + u} = \frac{-1 - u^2}{1 + u}
$$

分离变量：
$$
\frac{1 + u}{1 + u^2} \, du = -\frac{dx}{x}
$$

---

## 📐 三、积分与化简

左边拆分为：
$$
\frac{1 + u}{1 + u^2} = \frac{1}{1 + u^2} + \frac{u}{1 + u^2}
$$

积分：
$$
\int \left( \frac{1}{1 + u^2} + \frac{u}{1 + u^2} \right) du = \arctan u + \frac{1}{2} \ln(1 + u^2) + C_1
$$

右边：
$$
-\int \frac{dx}{x} = -\ln|x| + C_2
$$

合并常数，得：
$$
\arctan u + \frac{1}{2} \ln(1 + u^2) = -\ln|x| + C
$$

代回 $ u = \frac{y}{x} $：
$$
\arctan\left(\frac{y}{x}\right) + \frac{1}{2} \ln\left(1 + \frac{y^2}{x^2}\right) + \ln|x| = C
$$

注意：
$$
\frac{1}{2} \ln\left(1 + \frac{y^2}{x^2}\right) + \ln|x| = \frac{1}{2} \ln\left(\frac{x^2 + y^2}{x^2}\right) + \ln|x| = \frac{1}{2} \ln(x^2 + y^2) - \ln|x| + \ln|x| = \frac{1}{2} \ln(x^2 + y^2)
$$

因此：
$$
\arctan\left(\frac{y}{x}\right) + \frac{1}{2} \ln(x^2 + y^2) = C
$$

## 🎯 四、应用初始条件 $ y(1) = 0 $

代入 $ x = 1, y = 0 $：
- $ \arctan(0) = 0 $
- $ \ln(1^2 + 0^2) = \ln 1 = 0 $

所以 $ C = 0 $

最终方程：
$$
\arctan\left(\frac{y}{x}\right) + \frac{1}{2} \ln(x^2 + y^2) = 0
$$

两边乘以 2：
$$
\boxed{\ln(x^2 + y^2) + 2\arctan\left(\frac{y}{x}\right) = 0}
$$

✅ 与您给出的答案完全一致。

---

## 📌 五、几何意义补充

该曲线满足：**任意点处的切线与 $ y $ 轴交点的纵坐标 = 法线与 $ x $ 轴交点的横坐标**。

这是一个**对称性条件**，导致曲线具有特定的旋转对称性质。事实上，该隐式方程在极坐标下可进一步简化：

令 $ x = r\cos\theta, y = r\sin\theta $，则：
- $ \frac{y}{x} = \tan\theta \Rightarrow \arctan(y/x) = \theta $（在第一象限）
- $ x^2 + y^2 = r^2 $

方程变为：
$$
\ln(r^2) + 2\theta = 0 \Rightarrow 2\ln r + 2\theta = 0 \Rightarrow \ln r + \theta = 0 \Rightarrow r = e^{-\theta}
$$

这是一条**对数螺线**（logarithmic spiral）！

且由 $ y(1) = 0 \Rightarrow (x,y) = (1,0) \Rightarrow r=1, \theta=0 $，满足 $ r = e^{-\theta} $。

## ✅ 最终结论

您的解析无懈可击，答案正确。该曲线是一条**对数螺线**，其隐式方程为：

$$
\boxed{\ln(x^2 + y^2) + 2\arctan\left(\frac{y}{x}\right) = 0}
$$
