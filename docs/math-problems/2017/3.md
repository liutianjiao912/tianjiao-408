您的分析非常准确，对于每个选项的判断都基于正确的数学逻辑。我们来详细解析一下为什么选项 D 是正确的，并且其他选项为何不成立。

### A. 若 $\lim_{n \to \infty} \sin x_n = 0$，则 $\lim_{n \to \infty} x_n = 0$

这个命题是错误的。正如您所指出的，取 $x_n = 2n\pi$（其中 $n$ 是整数），则 $\sin x_n = \sin(2n\pi) = 0$ 对所有 $n$ 成立，但是 $\lim_{n \to \infty} x_n = \infty$ 而不是 $0$。这是因为 $\sin x$ 函数在每个周期内都会经过 $0$ 点，所以即使 $\sin x_n$ 的极限为 $0$，$x_n$ 的极限可以是任意值甚至是无穷大。

### B. 若 $\lim_{n \to \infty} (x_n + \sqrt{x_n^2 + 1}) = 0$，则 $\lim_{n \to \infty} x_n = 0$

这个命题也是错误的。考虑当 $x_n$ 接近负无穷时，$\sqrt{x_n^2 + 1}$ 接近 $|x_n|$。在这种情况下，$x_n + \sqrt{x_n^2 + 1}$ 接近 $0$，因为正项和负项几乎抵消了对方。然而，这并不意味着 $x_n$ 必须趋向于 $0$。例如，如果 $x_n = -n$，那么随着 $n$ 增加，$x_n + \sqrt{x_n^2 + 1}$ 会接近 $0$，但 $x_n$ 却趋向于负无穷。

### C. 若 $\lim_{n \to \infty} (x_n + |x_n|) = 0$，则 $\lim_{n \to \infty} x_n = 0$

这个命题同样是错误的。如果 $x_n < 0$，那么 $x_n + |x_n| = x_n - x_n = 0$ 对所有这样的 $x_n$ 都成立。因此，即使 $x_n$ 不趋向于 $0$（比如 $x_n = -1$），表达式 $x_n + |x_n|$ 的极限也可以是 $0$。

### D. 若 $\lim_{n \to \infty} x_n^2 = 0$，则 $\lim_{n \to \infty} x_n = 0$

这个命题是正确的。由于 $x_n^2$ 总是非负的，并且它的极限为 $0$ 意味着 $x_n$ 必须无限接近于 $0$。根据极限的定义，如果 $x_n^2$ 的极限为 $0$，那么对于任何 $\epsilon > 0$，存在一个 $N$ 使得对于所有的 $n > N$，有 $|x_n^2| < \epsilon$。这意味着 $-\sqrt{\epsilon} < x_n < \sqrt{\epsilon}$，从而 $x_n$ 必须趋向于 $0$。

综上所述，**正确答案是 D**。这个结论直接来自于实数平方根函数的连续性和极限的基本性质。如果您有任何疑问或者需要进一步解释，请随时告诉我！
