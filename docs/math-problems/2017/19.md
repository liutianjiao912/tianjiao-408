# 2017 数学二真题 · 第19题

您的解析要点基本正确，尤其是对第（I）问的处理非常到位。但第（II）问的逻辑略显模糊，**关键在于如何严格构造辅助函数并应用罗尔定理两次**。下面我们给出**完整、严谨的证明**，符合考研数学评分标准（参考知识库 [ty-reference](1)）。

---

## 🔍 一、已知条件整理

- $ f(x) \in C^2[0,1] $（在 $[0,1]$ 上具有二阶连续导数）
- $ f(1) > 0 $
- $ \lim_{x \to 0^+} \frac{f(x)}{x} = L < 0 $

由极限定义，存在 $ \delta > 0 $，使得当 $ 0 < x < \delta $ 时，
$$
\frac{f(x)}{x} < \frac{L}{2} < 0 \Rightarrow f(x) < 0
$$
（因为 $ x > 0 $）

此外，由于 $ f $ 在 $[0,1]$ 上二阶可导，故在 $ x=0 $ 处右连续，且
$$
\lim_{x \to 0^+} f(x) = 0 \quad \text{（否则 } \frac{f(x)}{x} \text{ 无有限极限）}
$$
因此可定义 $ f(0) = 0 $，使 $ f \in C[0,1] $。

---

## ✅ (I) 证明：$ f(x) = 0 $ 在 $ (0,1) $ 内至少有一个实根

- 由上述，存在 $ x_1 \in (0, \delta) \subset (0,1) $，使得 $ f(x_1) < 0 $
- 又 $ f(1) > 0 $
- $ f $ 在 $[x_1, 1] \subset [0,1] $ 上连续

由**介值定理**，存在 $ \xi \in (x_1, 1) \subset (0,1) $，使得 $ f(\xi) = 0 $

✅ **(I) 得证**。

---

## 🧠 (II) 证明：方程  
$$
f(x)f''(x) + [f'(x)]^2 = 0
$$  
在 $ (0,1) $ 内至少有两个不同实根

### Step 1：构造辅助函数

令：
$$
F(x) = f(x) f'(x)
$$

则：
$$
F'(x) = f'(x) f'(x) + f(x) f''(x) = [f'(x)]^2 + f(x) f''(x)
$$

这正是题目中方程的左边！

因此，只需证明：**$ F'(x) = 0 $ 在 $ (0,1) $ 内至少有两个不同实根**。

### Step 2：寻找 $ F(x) $ 的三个零点（或两个零点 + 端点性质）

我们已知：

- $ f(0) = 0 $（由极限存在且有限）
- 存在 $ a \in (0,1) $，使得 $ f(a) = 0 $（由 (I)）
- $ f(1) > 0 $

考虑 $ F(x) = f(x) f'(x) $：

- 在 $ x = 0 $：$ F(0) = f(0) f'(0) = 0 \cdot f'(0) = 0 $
- 在 $ x = a $：$ F(a) = f(a) f'(a) = 0 \cdot f'(a) = 0 $

所以，**$ F(x) $ 在 $ [0, a] $ 上满足 $ F(0) = F(a) = 0 $**

由**罗尔定理**，存在 $ \xi_1 \in (0, a) \subset (0,1) $，使得：
$$
F'(\xi_1) = 0
$$

但这只给出一个根。我们需要**第二个根**。

### Step 3：分析 $ f(x) $ 在 $ (a, 1] $ 的行为

- $ f(a) = 0 $，$ f(1) > 0 $
- 若 $ f(x) \equiv 0 $ on $ [a,1] $，则 $ f(1) = 0 $，矛盾
- 故存在 $ x_2 \in (a,1) $，使得 $ f(x_2) > 0 $

由于 $ f $ 连续，且 $ f(a) = 0 $，$ f(x_2) > 0 $，则 $ f $ 在 $ (a, x_2) $ 上**从 0 增至正数**，故存在局部最小值或单调递增。

关键观察：**$ f(x) $ 在 $ (0,1) $ 内变号**（负 → 0 → 正），因此 $ f'(x) $ 必须**至少两次变号**，或 $ f $ 有极值点。

更严谨地：

- 在 $ (0, a) $：$ f(x) < 0 $，$ f(a) = 0 $ ⇒ $ f $ 从负增至 0 ⇒ 存在 $ c \in (0,a) $ 使 $ f'(c) > 0 $
- 在 $ (a,1) $：$ f(a) = 0 $，$ f(1) > 0 $ ⇒ $ f $ 从 0 增至正 ⇒ 存在 $ d \in (a,1) $ 使 $ f'(d) > 0 $

但更重要的是：**$ f(x) $ 在 $ x=0^+ $ 附近为负，在 $ x=a $ 处为 0，在 $ x=1 $ 处为正**，所以 $ f $ 至少有一个**极小值点**或**极大值点**。

然而，最简洁的方法是：

### Step 4：利用 $ F(x) $ 在 $ x=1 $ 处的符号

考虑 $ F(1) = f(1) f'(1) $

- 若 $ f'(1) = 0 $，则 $ F(1) = 0 $，于是 $ F(a) = F(1) = 0 $，在 $ (a,1) $ 内存在 $ \xi_2 $ 使 $ F'(\xi_2) = 0 $
- 若 $ f'(1) \ne 0 $，则 $ F(1) \ne 0 $

但我们可以考虑**$ f(x) $ 在 $ (a,1) $ 内是否取得极值**。

由于 $ f(a) = 0 $，$ f(1) > 0 $，且 $ f \in C^2 $，若 $ f $ 在 $ (a,1) $ 内单调递增，则 $ f'(x) > 0 $，此时 $ F(x) = f(x) f'(x) > 0 $ for $ x \in (a,1) $

但注意：**$ F(a) = 0 $，$ F(x) > 0 $ on $ (a,1) $**，无法直接用罗尔定理。

### ✅ 更优策略：考虑 $ f(x) $ 在 $ (0,a) $ 和 $ (a,1) $ 各有一个极值点

由于：
- $ \lim_{x \to 0^+} f(x) = 0^- $，$ f(a) = 0 $，且 $ f(x) < 0 $ on $ (0,a) $
  ⇒ $ f $ 在 $ (0,a) $ 内必有**极小值点** $ c \in (0,a) $，使得 $ f'(c) = 0 $
- $ f(a) = 0 $，$ f(1) > 0 $，且 $ f(x) > 0 $ near 1
  ⇒ $ f $ 在 $ (a,1) $ 内必有**极小值或单调增**，但若单调增，则无驻点

但注意：**在 $ x = c \in (0,a) $，有 $ f'(c) = 0 $**，故：
$$
F(c) = f(c) f'(c) = f(c) \cdot 0 = 0
$$

而我们已有：
- $ F(0) = 0 $
- $ F(c) = 0 $，$ c \in (0,a) $
- $ F(a) = 0 $

于是：

- 在 $ [0, c] $：$ F(0) = F(c) = 0 $ ⇒ 存在 $ \xi_1 \in (0,c) $，$ F'(\xi_1) = 0 $
- 在 $ [c, a] $：$ F(c) = F(a) = 0 $ ⇒ 存在 $ \xi_2 \in (c,a) $，$ F'(\xi_2) = 0 $

因此，**$ F'(x) = 0 $ 在 $ (0,a) \subset (0,1) $ 内至少有两个不同实根**

✅ **(II) 得证**。

---

## 📌 补充说明：为何存在极小值点 $ c \in (0,a) $？

- $ f(x) < 0 $ on $ (0,a) $，$ f(0) = f(a) = 0 $
- 由**闭区间上连续函数的最值定理**，$ f $ 在 $ [0,a] $ 上取得最小值
- 最小值不可能在端点（因 $ f(0)=f(a)=0 $，而内部 $ f(x) < 0 $）
- 故存在 $ c \in (0,a) $，使得 $ f(c) = \min_{[0,a]} f(x) < 0 $
- 由于 $ f \in C^2 $，在内点 $ c $ 处可导，且为极值点 ⇒ $ f'(c) = 0 $

---

## ✅ 最终结论

(I) 由介值定理，$ f(x) = 0 $ 在 $ (0,1) $ 内至少有一个实根；  
(II) 构造 $ F(x) = f(x)f'(x) $，利用 $ f $ 在 $ [0,a] $ 上的极小值点及端点零点，应用罗尔定理两次，得 $ F'(x) = 0 $ 至少有两个不同实根。

---

## ✅ 完整答案：

**(I)** 由 $\lim_{x \to 0^+} \frac{f(x)}{x} < 0$ 知存在 $x_1 \in (0,1)$ 使 $f(x_1) < 0$，又 $f(1) > 0$，由介值定理，存在 $\xi \in (x_1,1) \subset (0,1)$ 使 $f(\xi) = 0$。

**(II)** 令 $F(x) = f(x)f'(x)$，则 $F'(x) = f(x)f''(x) + [f'(x)]^2$。  
由 (I) 知存在 $a \in (0,1)$ 使 $f(a) = 0$，又 $f(0) = 0$。  
因 $f(x) < 0$ on $(0,a)$，故 $f$ 在 $[0,a]$ 上有最小值点 $c \in (0,a)$，使得 $f'(c) = 0$，从而 $F(c) = 0$。  
于是 $F(0) = F(c) = F(a) = 0$，在 $(0,c)$ 和 $(c,a)$ 上分别应用罗尔定理，得 $F'(x) = 0$ 在 $(0,1)$ 内至少有两个不同实根。
