## 🔍 一、题目回顾

已知函数 $ f(x, y) $ 具有**一阶连续偏导数**，其全微分为：
$$
df(x, y) = y e^y \, dx + x(1 + y)e^y \, dy,
$$
且初始条件 $ f(0, 0) = 0 $，求 $ f(x, y) $。

---

## 🧠 二、理论依据

若 $ df = P(x, y)\,dx + Q(x, y)\,dy $ 是某函数 $ f(x, y) $ 的全微分，则：
- $ f_x = P $，$ f_y = Q $
- 且需满足**恰当微分条件**（因偏导连续，自动满足）：
  $$
  \frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}
  $$

验证（可选但推荐）：
- $ P = y e^y \Rightarrow P_y = e^y + y e^y = (1 + y)e^y $
- $ Q = x(1 + y)e^y \Rightarrow Q_x = (1 + y)e^y $
- 故 $ P_y = Q_x $，**确实是恰当微分**，存在原函数 $ f $。

---

## ✏️ 三、求解过程（与您一致）

### 步骤 1：由 $ f_x = y e^y $ 积分得
$$
f(x, y) = \int y e^y \, dx = x y e^y + C(y)
$$
（将 $ y $ 视为常数，积分对 $ x $）

### 步骤 2：对 $ y $ 求偏导
$$
f_y = \frac{\partial}{\partial y} \left( x y e^y + C(y) \right) = x e^y + x y e^y + C'(y) = x(1 + y)e^y + C'(y)
$$

但题设给出 $ f_y = x(1 + y)e^y $，故：
$$
C'(y) = 0 \Rightarrow C(y) = C \text{（常数）}
$$

### 步骤 3：利用初始条件 $ f(0, 0) = 0 $
$$
f(0, 0) = 0 \cdot 0 \cdot e^0 + C = C = 0
$$

因此：
$$
f(x, y) = x y e^y
$$

---

## ✅ 四、验证（可选）

计算其全微分：
- $ f_x = y e^y $
- $ f_y = x e^y + x y e^y = x(1 + y)e^y $

故：
$$
df = y e^y \, dx + x(1 + y)e^y \, dy
$$
与题设完全一致 ✅

---

## 📌 五、总结

本题是**由全微分反求原函数**的经典题型，关键在于：
1. 利用 $ f_x $ 积分得含 $ C(y) $ 的表达式；
2. 通过 $ f_y $ 确定 $ C(y) $；
3. 用初始条件定常数。

您的解答完美覆盖所有要点。
