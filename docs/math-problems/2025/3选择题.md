这是一道结合了**常微分方程解的性质**与**反常积分敛散性**的综合题目。解题的关键在于将反常积分的收敛性转化为微分方程特征根的实部符号问题。

以下是详细的解析步骤：

### 1. 写出特征方程
微分方程为 $y'' - 2ay' + (a+2)y = 0$。
对应的特征方程为：
$$r^2 - 2ar + (a+2) = 0$$

### 2. 分析积分收敛的充要条件
设特征方程的根为 $r_1, r_2$。微分方程的通解 $y(x)$ 由 $e^{r_1 x}$ 和 $e^{r_2 x}$ （或 $x e^{r_1 x}$，或 $e^{\alpha x}\cos(\beta x)$ 等形式）线性组合而成。
对于反常积分 $\int_{0}^{+\infty} y(x) dx$：
* 若 $y(x)$ 包含项 $e^{kx}$ （$k$ 为实数），则仅当 $k < 0$ 时积分收敛；若 $k \ge 0$，积分发散。
* 若 $y(x)$ 包含项 $e^{\alpha x}\cos(\beta x)$ （$\alpha, \beta$ 为实数），则仅当 $\alpha < 0$ 时积分收敛（指数衰减）；若 $\alpha \ge 0$，积分发散（震荡或趋于无穷）。

因此，要使**任一解**的反常积分均收敛，必须要求**特征方程的所有根的实部都严格小于 0**。即：
$$\text{Re}(r_1) < 0 \quad \text{且} \quad \text{Re}(r_2) < 0$$

### 3. 利用韦达定理（根与系数的关系）求解
对于二次方程 $r^2 + pr + q = 0$，两根实部均小于 0 的充要条件是系数满足：
$$p > 0 \quad \text{且} \quad q > 0$$
*(注：这对应于赫尔维茨稳定性判据在二阶系统中的应用)*

在本题中，特征方程为 $r^2 - 2ar + (a+2) = 0$。对应系数为：
* $p = -2a$
* $q = a+2$

根据上述条件，我们需要：
1.  **$p > 0$**：
    $$-2a > 0 \implies a < 0$$
2.  **$q > 0$**：
    $$a + 2 > 0 \implies a > -2$$

### 4. 综合结果
取两个不等式的交集，得到 $a$ 的取值范围为：
$$-2 < a < 0$$
即区间 $(-2, 0)$。

### 5. 分类讨论验证（可选，用于加深理解）
我们也可以根据判别式 $\Delta$ 分情况讨论，结果是一致的：
* **情形 1：实根 ($\Delta \ge 0$)**
    $\Delta = (-2a)^2 - 4(a+2) = 4(a^2 - a - 2) \ge 0 \implies a \ge 2$ 或 $a \le -1$。
    要实根均为负，需 $r_1+r_2 = 2a < 0$ 且 $r_1r_2 = a+2 > 0$。
    解得 $-2 < a < 0$。
    结合 $\Delta \ge 0$，得 $a \in (-2, -1]$。

* **情形 2：复根 ($\Delta < 0$)**
    $\Delta < 0 \implies -1 < a < 2$。
    此时根为 $r = a \pm i\beta$。实部为 $a$。
    要实部为负，需 $a < 0$。
    结合 $\Delta < 0$，得 $a \in (-1, 0)$。

求并集：$(-2, -1] \cup (-1, 0) = (-2, 0)$。

### 6. 结论
$a$ 的取值范围是 $(-2, 0)$。

**正确选项是 (C)。**
